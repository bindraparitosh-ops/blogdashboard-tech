Shopify OAuth — Connected Stores

Connect each store using the brand switcher. OAuth tokens are stored in your browser session — you'll need to reconnect after clearing cookies or on a new device.

Vercel Environment Variables Required
Shopify App Setup — For Each Store
1. Go to partners.shopify.com → Apps → Create app
2. App type: Custom app or Public app
3. Set redirect URL to: https://blogdashboard-tech.vercel.app/api/callback?brand=ctrl8
4. Required scopes: write_content, read_content
5. Copy Client ID → add to Vercel as SHOPIFY_CTRL8_CLIENT_ID
6. Copy Client Secret → add to Vercel as SHOPIFY_CTRL8_CLIENT_SECRET
